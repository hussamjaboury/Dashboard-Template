# Dashboard Project

## Project Description
A dashboard project designed to showcase multiple pages such as Dashboard, Projects, Files, Friends, Profile, and Settings. Built with HTML and CSS, it uses Font Awesome icons and sample images, and features a responsive, easy-to-customize design.

## Features
- Organized pages (Dashboard, Projects, Files, Friends, Profile, Settings).
- Responsive design that adapts to different screen sizes.
- Reusable components and page layouts.
- Easy to modify and extend.

## How to Run
1. Open the `index.html` file in a web browser to view the project.
2. Edit the HTML and CSS files in the project folder to modify the UI.
3. Replace images or icons in the `imgs/` folder and update the references in the HTML files.
